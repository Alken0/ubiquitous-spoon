For Security Reasons:
Please put one movie + related files in one folder and nothing else
(It is possible to fetch other files in the folders as well!)
The recommended format for your video-files is m3u8 (but browser support exists e.g. for mp4 as well).
You might wanna use ffmpeg to convert the files yourself:
ffmpeg -i [input-file.mkv] -hls_list_size 0 [output].m3u8
